# Welcome to immunecube

이 페이지는 Nextra + Next.js App Router가 제대로 동작하는지 확인하기 위한 **샘플 MDX 페이지**입니다.

- MDX가 정상 렌더링됩니다.
- 사이드바에 `docs`가 자동으로 나타납니다.
- 상단 Navbar, Footer는 layout.tsx에서 설정한 값이 표시됩니다.

## 섹션 1

문서를 원하는 만큼 추가해 보세요.
